## Strict Data Privacy Guidelines
- **NEVER** include or commit any references to test files, the Public Inquiry, Eljamel, NHS Scotland, NHS Tayside, NHS HIS, GMC, RCS England, Circle Health Group, NES, or any related sensitive data.
- This applies to all code, comments, UI placeholders, console logs, and commit messages.
- Always use highly generic examples (e.g. "Acme Corp Document") when writing placeholder text or placeholder logic.
